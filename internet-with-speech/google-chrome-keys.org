Following are a list of Google Chrome keyboard commands.

| Shortcut Keys                                                                            | Description                                                                                                                                                                                                            |
|------------------------------------------------------------------------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *[[file:///jargon/a/alt-home.htm][Alt+Home]]*                                            | Open your [[file:///jargon/h/homepage.htm][homepage]].                                                                                                                                                                 |
| *Alt+Left Arrow*                                                                         | Back a page.                                                                                                                                                                                                           |
| *Alt+Right Arrow*                                                                        | Forward a page.                                                                                                                                                                                                        |
| *F11*                                                                                    | Display the current website in full-screen mode. Pressing F11 again will exit this mode.                                                                                                                               |
| *Esc*                                                                                    | Stop loading the page or a [[file:///jargon/d/download.htm][download]] from loading.                                                                                                                                   |
| *Ctrl+(- or +)*                                                                          | [[file:///jargon/z/zoom.htm][Zoom]] in or out of a page, "-" will zoom out and "+" will zoom in on the page.                                                                                                           |
| *Ctrl+1-8*                                                                               | Pressing [[file:///jargon/c/ctrl.htm][Ctrl]] and any number 1 through 8 moves to the corresponding tab in your tab bar.                                                                                                |
| *Ctrl+9*                                                                                 | Switch to last tab.                                                                                                                                                                                                    |
| *Ctrl+0*                                                                                 | Reset browser zoom to default.                                                                                                                                                                                         |
| *[[file:///jargon/c/ctrl-enter.htm][Ctrl+Enter]]*                                        | This combination is used to quickly complete an address. For example, type "computerhope" in the address bar and press Ctrl+Enter to get [[https://www.computerhope.com]].                                             |
| *Ctrl+Shift+Del*                                                                         | Open the /Clear browsing data/ window to quickly clear private data.                                                                                                                                                   |
| *Ctrl+Shift+B*                                                                           | Toggle the bookmarks bar between hidden and shown.                                                                                                                                                                     |
| *[[file:///jargon/c/ctrl-a.htm][Ctrl+A]]*                                                | Select everything on a page.                                                                                                                                                                                           |
| *[[file:///jargon/c/ctrl-d.htm][Ctrl+D]]*                                                | Add a [[file:///jargon/b/bookmark.htm][bookmark]] for the page currently opened.                                                                                                                                       |
| *[[file:///jargon/c/ctrl-f.htm][Ctrl+F]]*                                                | Open the "find" bar to search text on the current page.                                                                                                                                                                |
| *[[file:///jargon/c/ctrl-o.htm][Ctrl+O]]*                                                | Open a file in the browser.                                                                                                                                                                                            |
| *Ctrl+Shift+O*                                                                           | Open the /Bookmark manager/.                                                                                                                                                                                           |
| *[[file:///jargon/c/ctrl-h.htm][Ctrl+H]]*                                                | Open browser history in a new tab.                                                                                                                                                                                     |
| *[[file:///jargon/c/ctrl-j.htm][Ctrl+J]]*                                                | Display the downloads window.                                                                                                                                                                                          |
| *[[file:///jargon/c/ctrl-k.htm][Ctrl+K]]* or *[[file:///jargon/c/ctrl-e.htm][Ctrl+E]]*   | Moves your text [[file:///jargon/c/cursor.htm][cursor]] to the [[file:///jargon/o/omnibox.htm][omnibox]] so that you can begin typing your search [[file:///jargon/q/query.htm][query]] and perform a Google search.   |
| *[[file:///jargon/c/ctrl-l.htm][Ctrl+L]]*                                                | Move the cursor to the browser [[file:///jargon/a/addrebar.htm][address bar]] and highlight everything in it.                                                                                                          |
| *[[file:///jargon/c/ctrl-n.htm][Ctrl+N]]*                                                | Open New browser window.                                                                                                                                                                                               |
| *Ctrl+Shift+N*                                                                           | Open a new window in [[file:///jargon/i/incognito.htm][incognito (private) mode]].                                                                                                                                     |
| *[[file:///jargon/c/ctrl-p.htm][Ctrl+P]]*                                                | Print current page or [[file:///jargon/f/frame.htm][frame]].                                                                                                                                                           |
| *[[file:///jargon/c/ctrl-r.htm][Ctrl+R]] or F5*                                          | Refresh the current page or frame.                                                                                                                                                                                     |
| *[[file:///jargon/c/ctrl-s.htm][Ctrl+S]]*                                                | Opens the /Save As/ window to save the current page.                                                                                                                                                                   |
| *[[file:///jargon/c/ctrl-t.htm][Ctrl+T]]*                                                | Opens a new [[file:///jargon/t/tab.htm][tab]].                                                                                                                                                                         |
| *[[file:///jargon/c/ctrl-u.htm][Ctrl+U]]*                                                | View a web page's [[file:///jargon/s/source.htm][source code]].                                                                                                                                                        |
| *[[file:///jargon/c/ctrl-w.htm][Ctrl+W]]*                                                | Closes the current tab.                                                                                                                                                                                                |
| *Ctrl+Shift+W*                                                                           | Closes the currently selected window.                                                                                                                                                                                  |
| *Ctrl+Shift+T*                                                                           | This combination reopens the last tab you've closed. If you've closed multiple tabs, you can press this shortcut key multiple times to restore each of the closed tabs.                                                |
| *[[file:///jargon/c/ctrl-tab.htm][Ctrl+Tab]]*                                            | Moves through each of the open tabs going to the right.                                                                                                                                                                |
| *Ctrl+Shift+Tab*                                                                         | Moves through each of the open tabs going to the left.                                                                                                                                                                 |
| *Ctrl+Left-click*                                                                        | Open a link in a new tab in the background.                                                                                                                                                                            |
| *Ctrl+Shift Left-click*                                                                  | Open a link in a new tab and switch to the new tab.                                                                                                                                                                    |
| *Ctrl+Page Down*                                                                         | Open the browser tab to the right.                                                                                                                                                                                     |
| *Ctrl+Page Up*                                                                           | Open the browser tab to the left.                                                                                                                                                                                      |
| *[[file:///jargon/s/spacebar.htm][Spacebar]]*                                            | Moves down a page at a time.                                                                                                                                                                                           |
| *Shift+Spacebar*                                                                         | Moves up a page at a time.                                                                                                                                                                                             |
| *[[file:///jargon/h/homekey.htm][Home]]*                                                 | Go to top of page.                                                                                                                                                                                                     |
| *[[file:///jargon/e/endkey.htm][End]]*                                                   | Go to bottom of page.                                                                                                                                                                                                  |
| *Alt+Down Arrow*                                                                         | Display all previous text entered in a text box and available options on a [[file:///jargon/d/dropdm.htm][drop-down menu]].                                                                                            |
